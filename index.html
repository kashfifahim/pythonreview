Lists

Term Definition
List an ordered sequence of heterogeneous elements
list.append() use this function to add a single element to the end of a list; O(1), constant time.
list.insert() inserts elements to the list; O(n) time.
list.remove() removes the given element at a given index; O(n) time; if element does not exist, runtime error.
list.pop() removes the element at given index-O(k) time; if no index is given, removes last element--O(1) time;
list.reverse() reverses the list; O(n) time

List Slicing

Term Description
list[start:end] start and end indicate the starting and ending index of a list that is desired to be accessed
list[start:] all elements greter than start uptil the range
list[:end] all elements less than end uptil the range
list[:] all numbers within the range
list[start:stop:step] step specified the increment in the list indices
range In python, the range counts up to the second index given but never hits the index itself
del list[x:y:z] delete eleemnts from a list
list[-x] we can use negative numbers to begin indexing the list elements from the end
